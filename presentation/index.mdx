---

Exemplos js

old
```javascript
function useToggle(initialOn = false) {
  const [on, setOn] = useState(initialOn);
  const toggle = () => setOn(!on);
  return { on, toggle };
}
```

new

```javascript
function App() {
  const { on, toggle } = useToggle();
  return <button onClick={toggle}>{on ? 'on' : 'off'}</button>;
}
```
